1. combine meshes in stuff using meshlab, save as all.obj.
2. place the scene in data/ with the file structure:
    data
    ├─<scene_name>
      ├─stuff
        ├─all.obj
        ├─<id>_<stuff_type>.png
        ├─...
      ├─things
        ├─<id>_<thing_type>.obj
        ├─<id>_<thing_type>.png
        ├─...
2. create jobs using create_jobs.py
3. run all jobs
4. run combine.py
